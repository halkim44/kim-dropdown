# kim-dropdown
a module for creating dropdown menu.

## Prerequisites
an html element is needed for this module to work

```
<div class="dropdown">
    <div class="title">
        <i class="angle-right"></i> // optional
    </div>
        
    <div class="menu">
        <div class="option">option 1</div>
        <div class="option">option 2</div>
        ...
    </div>
</div>
```
